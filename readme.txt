＊紹介

Ctrl(Command)-Tab, Ctrl(Command)-Shift-Tab, Ctrl(Command)-PageDown,
Ctrl(Command)-PageUp によるタブの切り替え動作を、前後のタブにフォー
カスするのではなく、一覧からタブを選んで切り替えるように変更します。

初期状態では、最後にフォーカスした順でタブの一覧を表示するという、
WindowsでいうAlt-Tabのような動作になっています。この動作は変更可能で、
タブの表示順そのままで一覧を表示することも可能です。


マウスによる操作をメインにしている人はJoe4711氏のTabScrollerの方が
お勧めです。
http://www.krickelkrackel.de/tabscroller/tabscroller.htm

＊インストール手順

  +-----------------------------------------------------------------+
  | ※旧バージョンからの更新の場合、インストールを始める前に、必    |
  |   ず、次項の「アンインストール手順」に従って旧バージョンをア    |
  |   ンインストールしておいて下さい。旧バージョンのファイルが残っ  |
  |   ている場合、導入に失敗したり、ブラウザの動作に支障が出る場合  |
  |   があります。                                                  |
  +-----------------------------------------------------------------+

step1: ファイルのコピー
  stackstyletabs.xpiをブラウザのウィンドウにドロップすると、インストー
  ルが開始されます。
  途中、日本語の言語パックを登録するかどうかの確認があるので、必要に応じ
  て選択して下さい。

step2: ブラウザの再起動
  ファイルのコピーが終わったら、ブラウザを再起動します。
  コピーしたファイル群が、次回起動時にブラウザに登録されます。

＊アンインストールの手順


Preferences（設定） > Advanced（詳細） > Stack Style Tabs（スタック型
タブ）で「Uninstall（アンインストール）」と書かれたボタンをクリック
すると、自動でアンインストールが行われます。

なお、ファイルの削除は自動では行われません。表示されるメッセージに従っ
て、ブラウザを終了させた後に、指定されたファイルを手動で削除して下さい。


ブラウザ自体起動できないなどの緊急の場合には、以下の手順に従って、手動
でアンインストールを行って下さい。

  1. ブラウザを終了させる。簡易起動（高速起動）が有効になっている場合、
     タスクトレイのアイコンを右クリックして、そちらも終了させる。
  2. ユーザープロファイルディレクトリか、ブラウザをインストールした
     ディレクトリ（管理者権限でインストールした場合）の中にある
     /chrome/ フォルダから、 stackstyletabs.jar, overlayinfo ディレクトリ,
     chrome.rdf を削除する。
  3. 管理者権限でインストールした場合、installed-chrome.txt をメモ帳等
     で開き、以下の行を削除する。
     ・content,install,url,jar:resource:/chrome/stackstyletabs.jar!
                           /content/stackstyletabs/
     ・locale,install,url,jar:resource:/chrome/stackstyletabs.jar!
                           /locale/en-US/stackstyletabs/
     ・locale,install,url,jar:resource:/chrome/stackstyletabs.jar!
                           /locale/ja-JP/stackstyletabs/

この手順では設定情報などが残ったままになりますが、ブラウザを使用する上
では問題ありません。
